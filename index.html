Introduction

This privacy policy describes how GMGN一键备注 Chrome Extension ("we", "our", or "the extension") handles user information. We are committed to protecting your privacy and ensuring the security of any information that may be collected through our extension.

Information Collection and Use

Our extension interacts exclusively with the GMGN.ai website and only processes the following information:

  • Access tokens stored in your browser's local storage by GMGN.ai
  • Token addresses from the current URL
  • Wallet information and remarks as displayed on GMGN.ai

All of this information is processed locally in your browser and is only used to interact with the GMGN.ai website's features.

Data Storage

The extension does not store any personal data independently. It only reads and processes data that is already stored by the GMGN.ai website in your browser's local storage.

Data Transmission

All data transmission occurs exclusively between your browser and GMGN.ai servers. The extension does not transmit any data to other third-party servers.

Permissions

The extension requires the following permissions:

  • activeTab: To interact with the current GMGN.ai page
  • host permissions for gmgn.ai: To make API calls to GMGN.ai servers
